从零开始：Clash Verge 全平台上手安装与订阅导入指南

拓扑技术组 • 更新日期: 2026-06-15 • 预计阅读: 8 分钟

由于 Clash 原版核心在 2023 年相继停止维护，许多用户面临着代理软件失效或无新版本更新的尴尬局面。 在此背景下，基于 Tauri + Rust 编写的全新开源代理客户端 Clash Verge 迅速崛起，成为目前最主流的 Clash 系列替代工具。

Clash Verge 不仅拥有高度美观、现代的 UI 界面，而且原生支持 Mihomo (原 Clash Meta) 内核，对 Hysteria 2、VLESS Reality 等新一代高能混淆协议提供了极佳的兼容性。

为什么推荐使用 Mihomo (Meta) 内核？

Mihomo 内核是传统 Clash 内核的超集。不仅在稳定性上有所提升，更重要的是它支持最新的抗封锁协议。如果您的专线机场或节点支持 VLESS 协议，必须在 Clash Verge 的“内核设置”中切换至 Meta 内核才能正常运行。

一、Clash Verge 客户端的下载与安装

Clash Verge 官方源代码在 GitHub 开源分发。为了避免下载到被植入木马的恶意修改版，建议始终通过其官方开源发布渠道进行下载：

  • Windows 用户：根据 CPU 架构下载 .exe 或是免安装的 .zip 压缩包。
  • macOS 用户：根据芯片类型下载（Intel 芯片选 x64.dmg，苹果 M1/M2/M3 芯片选 aarch64.dmg）。
  • Linux 用户：支持使用 .deb 或是 .AppImage 文件进行配置。
# 常用包命名格式示例
# Windows 64位安装包
Clash.Verge_1.x.x_x64-setup.exe

# macOS Apple 芯片安装包
Clash.Verge_1.x.x_aarch64.dmg

二、获取并导入您的机场订阅链接

获取到代理链接后，您需要将其导入 Clash Verge。以下是标准的导入流程：

  1. 打开浏览器登录您的加速机场后台，复制对应的 Clash 订阅链接 (YAML 格式)。
  2. 启动 Clash Verge 客户端，在左侧导航栏点击进入 “订阅 (Profiles)” 选项卡。
  3. 在顶部输入框中粘贴您刚才复制的订阅 URL 链接，点击右侧的 “导入 (Import)” 按钮。
  4. 等待系统下载完成。下载成功后，会出现一个带彩色进度条的配置文件卡片，鼠标右键点击它并选择“使用 (Use)”，卡片边缘高亮即代表生效。
特别提醒：保护好您的订阅链接！

订阅链接包含了您的节点密钥，类似于您购买套餐的专属密码。请勿公开分享，或将订阅输入未知安全的第三方在线“订阅转换器”中，否则极易导致订阅额度被盗刷或账户被防火墙阻断。

三、启用系统代理与分流模式选择

导入配置文件后，我们需要将其置于“运行”状态。在左侧导航栏点击 “设置 (Settings)”：

1. 找到 “系统代理 (System Proxy)” 选项，并将其开关开启。此时，您电脑的所有流量就会按分流规则进行代理。

2. 为了获得最佳体验，请在客户端托盘或侧边栏确保工作模式为 “规则 (Rule)” 模式。在此模式下，Clash 将自动判断访问目标：如果是国内网银、微信或淘宝，则直接访问不走代理；如果是 Google, YouTube 或 Netflix，则自动通过代理节点访问。这不仅可以节省套餐流量，更能有效避免国内软件因 IP 异地异常登录而弹警告。

四、无法联网？排查常见错误

在上手使用中，经常有用户反映导入后“红圈报错”或“浏览器无法打开网页”。这通常是由以下原因导致的，您可以按此排查：

1. 系统时间不同步 (核心原因)

绝大多数代理协议（如 VMess/Trojan/VLESS）都对时钟同步有极其严格的要求。如果您的电脑系统时间与北京时间误差超过 90 秒，节点会报错 TIMEOUT 或握手失败。请进入 Windows 设置，点击“同步时钟”即可解决。

2. 虚拟网卡或代理冲突

如果在 Clash Verge 之前，您电脑运行过旧版 Clash for Windows 或其他代理网速工具，其残留的系统端口占用可能会导致新软件无法成功代理。请确保彻底关闭其他代理工具，并在“设置”中重置网络属性。
